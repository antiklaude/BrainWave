## Phase 1 — Learn (50 min)

**Why this matters:** TinkerCAD is fast but limited. Once you need precise tolerances, complex curves, or a design that needs to be edited after the fact, you need parametric CAD. Fusion 360 is the industry-standard tool for makers and engineers — it's free for personal use, and the skills translate directly to professional mechanical design. Learning it opens up functional part design: enclosures, mechanisms, tools, replacement parts.

- **Fusion 360 workspace:**
  - The *Browser* (left panel) shows components, bodies, sketches, and origins
  - The *Timeline* (bottom) records every operation in order — click any step to go back to that state
  - The *Design toolbar* (top) contains all modelling tools: Sketch, Solid, Surface, Sheet Metal
- **Sketch workflow** — all Fusion geometry starts with a 2D sketch:
  1. Select a plane or flat face as the sketch plane
  2. Create → Sketch (or press S for the shortcut panel)
  3. Draw geometry using lines, arcs, rectangles, circles
  4. Add dimensions (D for dimension tool) to constrain sizes
  5. Add geometric constraints (right-click → Sketch Constraints): parallel, perpendicular, concentric, coincident
  6. When the sketch is fully constrained (all lines black), finish the sketch
- **Extrude (E)** — select a closed sketch profile → Extrude → set distance. Operations: Join (add to existing body), Cut (subtract), New Body.
- **Fillet (F)** — rounds a sharp edge. Select edges, set radius. Essential for 3D printing — sharp internal corners are stress concentrations and can cause prints to crack under load.
- **Chamfer** — cuts a flat angled face on an edge (instead of a round fillet). Use on top edges of prints to avoid the "elephant foot" first layer artifact.
- **Parameters** — Modify → Change Parameters. Set named variables (e.g., `wall_thickness = 2mm`). Use parameter names in dimension dialogs instead of numbers. When you change `wall_thickness` to 3mm, all walls update. This is parametric design.
- **Export STL for printing** — right-click the body in the Browser → Save As Mesh → STL format. Select resolution (0.1mm chord deviation is fine for printing).

> **Key insight:** Spend the extra 2 minutes making your sketch fully constrained before extruding. Underconstrained sketches produce unpredictable results when you edit dimensions. Black lines = safe to extrude. Blue lines = fix the constraints first.

## Phase 2 — Do (50 min)

Design a wall-mount bracket with bolt holes:
1. Create a new design in Fusion 360
2. **Set up parameters** (Modify → Change Parameters):
   - `width = 60mm`
   - `height = 80mm`
   - `thickness = 4mm`
   - `hole_diameter = 4mm`
3. **Sketch on XZ plane:** draw a rectangle using `width × height` parameters, finish sketch
4. **Extrude** by `thickness` parameter → creates the bracket plate
5. **Sketch the bolt holes:** create a new sketch on the front face, draw 4 circles (diameter = `hole_diameter`) at the 4 corners, 8mm from each edge. Finish sketch.
6. **Extrude → Cut** through the bracket to create through-holes
7. **Fillet** all vertical edges with a 2mm radius
8. **Test the parametric behaviour:** Modify → Change Parameters → change `thickness` to 6mm. The bracket updates. Change `hole_diameter` to 5mm. The holes update.
9. Export as STL, open in Cura, verify the holes look correct in layer view
