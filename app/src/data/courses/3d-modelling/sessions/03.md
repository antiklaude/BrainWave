## Phase 1 — Learn (45 min)

**Why this matters:** Object Mode controls whole objects. Edit Mode controls the mesh itself — the actual geometry. All modelling happens in Edit Mode. Normals are invisible but critical: they determine which direction a surface faces, affecting how light hits it, whether it appears in renders, and how modifiers behave.

- **Tab** — toggle between Object Mode and Edit Mode. In Edit Mode, the selected object turns orange.
- **Select modes** — in Edit Mode, press 1 (vertex), 2 (edge), or 3 (face) to switch what you're selecting. These are different from the numpad.
- **Selecting** — left-click to select, Shift+click to add to selection, Alt+click to select a loop, A to select all, Alt+A to deselect all.
- **Extrude (E)** — extends selected geometry outward. The new geometry is connected to the original. Fundamental for growing a mesh from a base shape.
- **Loop Cut (Ctrl+R)** — adds an edge loop running around the mesh. Hover over the model to preview the direction, scroll to add multiple cuts, click to confirm, move to position.
- **Inset (I)** — creates a smaller face inside a selected face. Used to create panels, window frames, details.
- **Normals** — every face has a direction (outward-facing normal). This determines how light bounces off the surface. Inverted normals cause issues in renders and modifiers.
- **Face Orientation overlay** — in the Viewport Overlays (top-right button), enable "Face Orientation". Blue = correct (normal pointing outward). Red = flipped (normal pointing inward). In a clean model, everything should be blue.
- **Recalculate Normals (Shift+N)** — in Edit Mode, select all (A), then Shift+N to automatically recalculate all normals to point outward. Fixes most normal issues instantly.

> **Key insight:** If part of your model looks invisible or pitch black in a render, the normals are wrong. Enable Face Orientation overlay (look for red faces), select all in Edit Mode, press Shift+N. This one shortcut fixes the most common render mystery for beginners.

## Phase 2 — Do (45 min)

- Model a low-poly house from a cube:
  1. Start with a cube (Shift+A → Mesh → Cube)
  2. Tab into Edit Mode
  3. Select the top face, extrude up (E → Z) to create walls
  4. Add a loop cut (Ctrl+R) near the top to define the roof line
  5. Select the top edge loop and scale it (S → X) to create a pitched roof shape
  6. Enable Face Orientation overlay — verify all faces are blue
  7. If any red faces: select all (A) → Shift+N to fix
