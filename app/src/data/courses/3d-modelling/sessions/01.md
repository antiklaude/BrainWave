## Phase 1 — Learn (40 min)

**Why this matters:** Blender has one of the steepest learning curves of any creative software — not because it's complex, but because it's different from everything else. The first session is entirely about getting comfortable with the interface so it stops fighting you. Time spent here saves hours of frustration later.

- **Viewport navigation** — middle mouse button to orbit, scroll wheel to zoom, Shift+middle mouse to pan. On a laptop without a mouse: enable "Emulate 3 Button Mouse" in Preferences.
- **Numpad shortcuts** — critical muscle memory:
  - Numpad 1 — front view
  - Numpad 3 — right side view
  - Numpad 7 — top view
  - Numpad 5 — toggle perspective/orthographic
  - Numpad 0 — camera view
- **Main panels** — 3D Viewport (where you work), Properties (object/material/render settings on the right), Outliner (scene hierarchy top-right), Timeline (bottom, for animation).
- **Header bar** — the bar at the top of each panel shows context-specific tools and the current mode.
- **Workspaces** — tabs across the top (Layout, Modelling, Sculpting, etc.). Each workspace is a pre-configured layout for a different task.
- **Object Mode vs Edit Mode** — Tab key toggles between them. Object Mode: select/move/scale entire objects. Edit Mode: edit the mesh (vertices, edges, faces) of the selected object.

> **Key insight:** Blender's interface is modal — what your keyboard does depends entirely on where your mouse cursor is hovering. Before pressing a shortcut, make sure your mouse is over the correct panel. This rule explains 80% of "my shortcut isn't working" issues.

## Phase 2 — Do (40 min)

- Open Blender. Delete the default cube (X key → Delete)
- Add a UV Sphere (Shift+A → Mesh → UV Sphere)
- Practice all viewport navigation: orbit, zoom, pan, numpad views
- Add a plane (Shift+A → Mesh → Plane) and a cylinder
- Select each object individually (left-click), then select all (A key), then deselect all (Alt+A)
- Tab into Edit Mode on the cube — notice how the header changes, how the object turns orange
