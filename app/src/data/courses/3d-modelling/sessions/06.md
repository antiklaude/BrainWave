## Phase 1 — Learn (50 min)

**Why this matters:** Hard surface and organic modelling require completely different approaches. You can't model a human head the same way you model a crate. Subdivision Surface and Sculpt Mode are the tools that make smooth, organic shapes possible — characters, creatures, vehicles with curved surfaces, natural forms.

- **Subdivision Surface modifier** — adds geometry and smooths the result. Level 1–2 is the sweet spot for most work. Higher = smoother but heavier (more polygons = slower viewport and render). Add it as a modifier, not by actually subdividing — this keeps the model editable.
- **Supporting edge loops** — Subdivision Surface rounds everything. To keep edges sharp, add edge loops close to them (Ctrl+R). Close loop = sharp edge. Far apart = rounded edge. This is how you control the shape.
- **Shade Smooth** — right-click in Object Mode → Shade Smooth (or Shade Auto Smooth for better results with flat areas). This doesn't add geometry — it interpolates normals to create a smooth appearance. No polygon cost.
- **Sculpt Mode** — accessed from the Mode dropdown (or Ctrl+Tab). Key brushes for beginners:
  - *Grab* — moves geometry where you drag. Great for shaping overall form.
  - *Clay* — builds up volume. Use to add mass.
  - *Smooth* — evens out bumpy or jagged areas (hold Shift in other brushes to temporarily activate).
- **Dyntopo** — Dynamic Topology. Enables in Sculpt Mode (N panel or top header). Adds geometry dynamically as you sculpt — useful for adding fine detail where needed without subdividing the whole mesh first.

> **Key insight:** For organic modelling, always apply Shade Smooth before sculpting or adding Subdivision Surface. The smooth shading makes it much easier to see surface bumps and flow accurately. Without it, the faceted appearance hides problems.

## Phase 2 — Do (50 min)

- Model a simple creature head:
  1. Start with a UV Sphere (Shift+A → Mesh → UV Sphere, 32 segments)
  2. Apply Scale
  3. Add Subdivision Surface modifier (Level 2)
  4. Right-click → Shade Smooth
  5. Enter Sculpt Mode
  6. Use Grab to elongate the face, push in eye sockets, pull out a snout
  7. Use Clay to add cheekbones and brow ridges
  8. Use Smooth to even out rough areas
