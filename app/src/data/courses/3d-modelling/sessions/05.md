## Phase 1 — Learn (50 min)

**Why this matters:** Hard surface modelling — machines, vehicles, weapons, props, architecture — is one of the most in-demand 3D skills in games, film, and product visualisation. The techniques in this session (booleans, bevel, inset) are the core toolkit for creating the clean, mechanical look of manufactured objects.

- **Boolean modifier** — performs set operations between two meshes. The three modes:
  - *Union* — merges two overlapping objects into one
  - *Difference* — subtracts one object from another (creates holes and cavities)
  - *Intersect* — keeps only the overlapping volume
  For clean results: apply Booleans last, and use the "Exact" solver for complex meshes.
- **Ctrl+B (Bevel)** — bevels edges to create chamfers. Scroll the mouse wheel while bevelling to add more segments (more = rounder). The "Width" controls how wide the bevel spreads. Add a Bevel modifier instead of manual bevelling for non-destructive control.
- **Inset (I)** — creates a smaller inset face inside a selected face. Move the mouse to control the inset distance. Used for panel details, window recesses, grilles.
- **Loop cuts (Ctrl+R)** — add supporting edge loops near bevelled or sharp edges to control how Subdivision Surface rounds them. Edges with loops close together remain sharp; edges without loops round smoothly.
- **Box modelling workflow** — start with a primitive (cube, cylinder), extrude and loop cut into the basic shape, then add details with inset and boolean.

> **Key insight:** Booleans are power tools — they're fast but can leave messy topology (triangles, n-gons) at the cut edges. For portfolio work, clean up the topology after a boolean. For game assets, clean topology matters for performance. For quick visualisation, messy topology is often acceptable.

## Phase 2 — Do (50 min)

Model a game-ready wooden crate:
1. Start with a cube, scale to a non-square box shape (e.g., 1.5×1×1)
2. Apply Scale
3. Add horizontal and vertical inset panels on each face (select face → I → adjust → confirm)
4. Create bolt details: add small spheres positioned at panel corners, use Boolean (Union) to attach them
5. Add edge bevels (Ctrl+B with 2 segments) to all main edges
6. Check normals with Face Orientation overlay
