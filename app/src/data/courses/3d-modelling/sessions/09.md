## Phase 1 — Learn (45 min)

**Why this matters:** Image textures are great but they have limitations: they need a UV map, they have fixed resolution, and they tile visibly on large surfaces. Procedural textures are generated by math — they scale to any size, never tile visibly, and can be layered and combined to create unique surfaces that don't exist as photographs. Knowing procedural nodes makes you independent of texture libraries.

- **Noise Texture** — generates organic, cloud-like patterns. Controls: Scale (frequency), Detail (complexity), Roughness (how sharp or soft), Distortion (warps the pattern). Used for: organic surfaces, rust, dirt, clouds.
- **Voronoi Texture** — generates cell patterns (like soap bubbles or cracked mud). Controls: Scale, Randomness. Used for: stone, cracked surfaces, alien textures.
- **Wave Texture** — generates stripe or ring patterns. Band or Ring type. Controls: Scale, Distortion, Detail. Used for: wood grain, water ripples, layered stone.
- **ColorRamp** — remaps a texture's 0–1 value output into a colour gradient. Add colour stops, adjust positions. Used after any texture to control contrast and colour mapping.
- **Mix Shader / Mix Color** — blends two shaders or colours together. Factor = 0 shows the first, Factor = 1 shows the second. Can be driven by another texture to mix based on surface variation.
- **Texture Coordinate + Mapping nodes** — Texture Coordinate provides the base coordinate input (Object, UV, Generated). Mapping node allows scaling, rotating, and translating the texture. Always add these to the start of any procedural texture chain if you want easy control.
- **Node Groups (Ctrl+G)** — select multiple nodes → Ctrl+G to group them. The group becomes a reusable node that can be named and reused across materials. Edit the group by pressing Tab inside it. Add input/output sockets via the Group Input/Output nodes.

> **Key insight:** Start every procedural material by asking "what natural phenomenon am I simulating?" — rock forms in layers (Wave + distortion), wood has grain rings (Wave + Noise for variation), rust is random variation on metal (Noise + ColorRamp + Mix). Then build the math that describes that pattern.

## Phase 2 — Do (45 min)

Build two procedural materials from scratch:

**Material 1 — Procedural Rock:**
1. Add Noise Texture → ColorRamp → Principled BSDF Base Color
2. Adjust ColorRamp stops to grey → dark grey (rock colours)
3. Add a second Noise Texture → ColorRamp → Roughness input (rocky areas are rougher)
4. Optional: add a third Noise Texture for a subtle normal variation

**Material 2 — Procedural Wood Grain:**
1. Add Wave Texture (set to Bands, increase Scale to ~20) → ColorRamp → Base Color
2. Adjust ColorRamp to light brown → dark brown
3. Add a Noise Texture → connect to the Wave Texture's Distortion input to break up the perfect rings
4. Group both materials' node trees (Ctrl+G) and name them "Rock Material" and "Wood Material"
5. Apply both to two separate objects and compare
