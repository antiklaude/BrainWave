## Phase 1 — Learn (40 min)

**Why this matters:** Typography is 90% of design. Most websites are mostly text. The choice of typeface, size, weight, and spacing communicates your brand personality before anyone reads a single word. Bad typography makes even good content look unprofessional.

- **Typeface categories**:
  - *Serif* — Times New Roman, Georgia, Playfair Display. Traditional, editorial, trustworthy. Great for long-form reading and luxury brands.
  - *Sans-serif* — Inter, Helvetica, Roboto. Modern, clean, versatile. The default for digital interfaces.
  - *Monospace* — Courier, JetBrains Mono. Code editors, technical writing.
  - *Display* — decorative, only for headlines at large sizes. Never for body text.
- **Typographic hierarchy** — size + weight create visual order. H1 (large/bold) → H2 (medium/semibold) → body (regular, comfortable size). The hierarchy tells the reader what to read in what order.
- **Font pairing rule** — use maximum 2 typefaces. One with personality (display, serif) for headings. One neutral and readable for body. Contrast is key: pair a serif with a sans-serif.
- **Kerning** — spacing between specific letter pairs. Well-kerned type looks balanced; bad kerning makes some letters look too close or too far apart.
- **Leading (line-height)** — vertical space between lines. 1.4–1.6 of the font size is the sweet spot for body text. Too tight makes reading exhausting; too loose makes text feel disconnected.
- **Tracking (letter-spacing)** — uniform spacing across all letters. Use slightly positive tracking on uppercase text (it's easier to read). Avoid negative tracking on body text.

> **Key insight:** When in doubt, use a classic sans-serif (Inter, Roboto, Source Sans) for body text. The most readable designs use boring type for body text and interesting type only for headings. Interesting body type is usually unreadable body type.

## Phase 2 — Do (40 min)

- On Google Fonts, find 2 font pairings that work together — one for headings, one for body text. Use the pairing preview to test them.
- Using any tool (Figma, Canva, even Google Slides), create a simple layout showing H1/H2/H3/body hierarchy using your chosen fonts
- Set body text at 16px with 1.5 line-height — how does it feel compared to 14px with 1.2 line-height?
- Check: does your heading font communicate the personality of the brand from Session 01?
